📁 **정리 목적**  
- 본 저장소는 프로젝트별 코드 흐름을 체계적으로 정리하기 위해 제작되었습니다.  
- 공개가 어려운 경우에도 주요 설계와 흐름에 대한 문서화 및 요약을 제공하고자 노력하고 있습니다.
